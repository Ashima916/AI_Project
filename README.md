🤖 AI Resume & Cover Letter Generator

An AI-powered web application that generates ATS-friendly resumes and professional cover letters based on user inputs. Built using Google Gemini API and deployed via Google Colab with an interactive Gradio interface.

🚀 Features
✨ Generate tailored resumes and cover letters instantly
📄 ATS-friendly formatting for better job screening results
🧠 Powered by Google Gemini 2.5 Flash (GenAI)
🎯 Custom inputs: name, role, skills, experience, job description
📥 Export documents in multiple formats:
.docx (Word format)
.pdf
🌐 User-friendly interface using Gradio

🛠️ Tech Stack
Python
Google Gemini API (GenAI)
Gradio (UI Framework)
python-docx (Word file generation)
ReportLab (PDF generation)
Google Colab (Development & Deployment)

⚙️ How It Works
User enters details such as:
Name
Job role
Skills
Experience
Job description
The app sends this data to the Gemini API
AI generates:
A structured, ATS-friendly resume
A customized cover letter
User downloads files in .docx or .pdf format
